# Chef Claude 🍳🤖

An AI-powered recipe generator built with React and Vite.  
Users can add ingredients they have at home and generate recipes using AI.

---

## 🚀 Features

- Add ingredients dynamically
- Generate AI recipes from available ingredients
- Responsive React UI
- Built with Vite for fast performance
- Uses Hugging Face API
- Component-based architecture

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- Hugging Face Inference API
